# AI Trip Planner 🌍✈️

A full-stack AI-based trip planner web app that helps users plan travel itineraries using AI.

## 🚀 Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **AI API**: OpenAI / Gemini (to be integrated)
- **Database**: (Planned – MongoDB or MySQL)

- [ ] Backend setup in progress
